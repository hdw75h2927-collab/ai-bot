# 🤖 Telegram AI Bot — Умный ассистент с нейросетью

> **Актуально на май 2026 года** — использует новую функцию Telegram **Chat Automation**

---

## Что это?

AI-бот, который подключается к вашему профилю Telegram и **автоматически отвечает на сообщения за вас** с помощью нейросети. Бот работает 24/7 и поддерживает русский язык.

## Возможности

- ✅ Ответы на сообщения с помощью нейросети (GigaChat / OpenRouter)
- ✅ Работает через функцию **Chat Automation** (Telegram, май 2026)
- ✅ Поддержка русского и других языков
- ✅ Запоминает контекст разговора
- ✅ Бесплатный хостинг (Render.com)
- ✅ Работает 24/7 без перерывов

## Быстрый старт (5 шагов)

### Шаг 1: Создайте бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Укажите имя бота (например: "Мой AI Ассистент")
4. Укажите username бота (например: `my_ai_assistant_bot`) — обязательно на `_bot`
5. Скопируйте **токен бота** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получите бесплатный API-ключ для AI

**Вариант А — GigaChat (Sber)** — рекомендуется, лучше понимает русский:
1. Перейдите на https://developers.sber.ru/products/gigachat
2. Зарегистрируйтесь (можно через Сбер ID или email)
3. Получите **Client ID** и **Client Secret**
4. Используйте их для получения access token

**Вариант Б — OpenRouter** — бесплатные модели:
1. Перейдите на https://openrouter.ai/keys
2. Зарегистрируйтесь (можно через Google/GitHub)
3. Создайте API-ключ (бесплатно)

### Шаг 3: Запустите бота

**Вариант А — Локально (для теста):**
```bash
# Установите зависимости
pip install -r requirements.txt

# Создайте файл .env и заполните его
cp .env.example .env
# Отредактируйте .env — вставьте токен и API-ключ

# Запустите
python bot.py
```

**Вариант Б — Бесплатный облачный хостинг (24/7):**

#### Render.com (бесплатно, рекомендуется):

1. Зарегистрируйтесь на https://render.com (через GitHub)
2. Создайте новый **Web Service**
3. Подключите GitHub-репозиторий с этим кодом
4. Укажите:
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python bot.py`
5. В разделе **Environment** добавьте переменные:
   - `BOT_TOKEN` — токен от @BotFather
   - `GIGACHAT_API_KEY` или `OPENROUTER_API_KEY` — ключ AI
6. Нажмите **Create Web Service**

> Бесплатный тариф Render даёт 750 часов в месяц (достаточно для 24/7 работы одного бота)

### Шаг 4: Включите Business Mode для бота

1. В Telegram найдите **@BotFather**
2. Отправьте `/mybots`
3. Выберите вашего бота
4. Нажмите **Bot Settings** → **Business Mode** → **Turn On**

### Шаг 5: Подключите Chat Automation

1. Откройте **Settings** (Настройки) в Telegram
2. Перейдите в **Chat Automation** (Автоматизация чатов)
   - *Если раздел не виден — обновите Telegram до версии 12.7.0+*
3. Введите username вашего бота (например: `@my_ai_assistant_bot`)
4. Выберите, какие чаты автоматизировать:
   - **All 1-to-1 Chats** — все личные сообщения
   - **Only Selected Chats** — только выбранные чаты
   - Настройте исключения (например, "Exclude contacts" — не отвечать контактам)
5. Готово! Бот теперь отвечает за вас 🤖

## Как проверить, что всё работает?

1. Напишите своему боту в личку какое-нибудь сообщение
2. Бот должен ответить с помощью AI
3. Если отвечает заглушка — проверьте, что API-ключ указан правильно
4. Команда `/status` покажет текущий статус AI

## Решение проблем

### Бот не отвечает
- Проверьте, что токен указан правильно
- Убедитесь, что бот запущен (логи в консоли)

### AI не работает (ответы-заглушки)
- Проверьте API-ключ в `.env`
- Убедитесь, что у провайдера есть кредиты

### Chat Automation не виден в настройках
- Обновите Telegram до версии 12.7.0 или выше
- Проверьте на другом устройстве

### Бот отвечает не от моего имени
- Убедитесь, что включили **Business Mode** в @BotFather
- Проверьте, что бот подключён в **Chat Automation**
